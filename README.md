## PURPOSE

Arduino code for Humidor app.

Posts the humidity / temperature sensor readings to the server.

## USAGE INSTRUCTIONS

Setup Arduino Makefile (see [Arduino-Makefile](http://github.com/sudar/Arduino-Makefile) )

make

make upload
